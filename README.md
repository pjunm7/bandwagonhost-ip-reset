## 换完IP之后要做的事

换好IP不等于万事大吉，有几件事要跟上：

**更新DNS解析**：如果你的域名解析到了原来的IP，换完之后要去域名服务商把DNS记录改成新IP，否则网站继续指向一个死地址。

**更新SSH配置**：客户端的known_hosts里可能记录了旧IP，连接时可能报警告，删掉旧记录重新连就好。

**服务器上的静态IP配置**：如果你的VPS系统配置了静态IP而不是DHCP，需要手动把配置文件里的IP地址改成新的。大部分人用的是默认配置，这步不需要。

还有一点：换好之后发现新IP还是用不了，可以直接回复工单告知客服。搬瓦工保证换来的新IP通过了全球连通性验证，如果确实有问题他们会处理。
